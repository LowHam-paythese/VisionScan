About This Project

VisionScan is a high-performance network discovery tool specifically engineered for identifying ONVIF-compliant security cameras. Built with a Node.js backend and a modern Vanilla JS/CSS frontend, it leverages low-level system utilities to map network topologies and detect specialized hardware.

Technical Highlight: This project explores full-stack integration of system-level scanning tools (Nmap/ARP) into a responsive web interface, featuring a custom implementation of Digest Authentication for camera configuration.
